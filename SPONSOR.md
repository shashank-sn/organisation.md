# support organisation.md

**organisation.md** is an open-source project — fork it, run it, build on it. if it saves your team time, consider sponsoring.

---

## sponsorship tiers

### one-time — $5+
a single contribution to say thanks. no commitment, no recurring payments. every bit helps with hosting costs, testing, and new features.

### monthly — $5/month
you rely on organisation.md and want to see it improve. you get:
- name listed on the project README
- priority consideration for feature requests
- early access to beta releases

### monthly — $15/month
you use organisation.md daily and want it actively maintained. you get everything above plus:
- direct access to the maintainer via github issues
- vote on the roadmap and priority of upcoming features
- private channel on the community discord

### monthly — $50/month
your team depends on organisation.md. you get everything above plus:
- your logo on the project README and site
- dedicated support with 24-hour response time
- custom feature development (within reason)
- onboarding session for your team

---

## what your sponsorship funds

- **development** — new tools, better parsing, bug fixes
- **infrastructure** — ci/cd pipeline, testing, documentation site
- **community** — issue triage, pr review, discussions
- **time** — maintainer availability to answer questions and help with setups

all features ship to everyone equally — no paid-only features. sponsoring funds the maintainer's time, not access to the code.
