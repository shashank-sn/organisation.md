# CONTEXT

This directory holds auxiliary context files that support the main `organisation.md` file. Agents can read these for deeper understanding of specific areas.

## Files

- `projects.md` — Detailed project information
- `architecture.md` — System architecture and technical decisions
- `people.md` — Team member details and expertise

## Usage

MCP tools like `list_context_files` and `search_context` include these files automatically.
